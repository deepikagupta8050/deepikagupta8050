##  About Me

I'm a Computer Science undergraduate specializing in AI & ML, and I build **full-stack, production-grade web applications** — from the database schema up to the pixel on screen. What draws me to this field is the combination of two things I genuinely enjoy: **engineering systems that scale**, and **applying AI to solve problems that matter**.

During my research internship at the **Saha Institute of Nuclear Physics, Kolkata**, I worked on a real, unsolved problem — detecting biomolecular condensates in cancer cell microscopy images. I trained and evaluated **5 U-Net architectures on 6,000+ real-world images**, improving on an existing metaheuristic baseline by **15–20% precision**. That experience taught me how to work independently on ambiguous, open-ended problems and deliver a measurable result — a mindset I now bring to every project I build.

Outside of research, I've shipped two full-stack AI products end-to-end — from system design and API architecture to deployment — and I'm currently sharpening my skills in **system design** and **scalable backend architecture** as I look for my first full-time role.

## Featured Work

### CodeSense AI — *AI-Powered Code Review Platform*
An automated code review tool that helps development teams catch bugs and quality issues before they reach production.
- Built a full-stack platform using **React, Next.js, TypeScript, Node.js, and Express**, with an SEO-optimized frontend
- Integrated **Groq LLM** for real-time automated bug detection, code analysis, and quality scoring
- Automated pull request and repository reviews at scale via **GitHub API** integration
- Shipped with a **CI/CD pipeline** for production-grade, reliable deployment

**Stack:** `React` `Next.js` `TypeScript` `Node.js` `Express` `Groq LLM` `GitHub API`
🔗 [Repository](https://github.com/deepikagupta8050/codesense-ai) · [Live Demo](https://codesense-ai-ten.vercel.app/)

### ScholarAI — *AI-Powered Research Intelligence Platform*
A research assistant platform that helps academics analyze papers faster and identify genuine research gaps.
- Built with **Next.js, TypeScript, and FastAPI**, deployed on Vercel and Render
- Automated paper analysis, novelty scoring, and research-gap detection using **LLM prompt engineering** (Groq API)
- Designed multi-endpoint **REST APIs** for PDF extraction, AI orchestration, and multi-paper analysis
- Implemented **Supabase** authentication, PostgreSQL storage, and a **D3.js** graph for paper visualization

**Stack:** `Next.js` `TypeScript` `FastAPI` `Supabase` `PostgreSQL` `D3.js`
🔗 [Repository](https://github.com/deepikagupta8050/ScholarAI) · [Live Demo](https://scholar-ai-rust.vercel.app/)

<br>

### Deep Learning-Based Puncta Segmentation — *Research Project*
A deep learning pipeline built during my internship at SINP Kolkata to detect biomolecular condensates in cancer cell imaging.
- Developed and evaluated **5 U-Net variants** for image segmentation using **PyTorch**
- Trained on **6,000+ real-world microscopy images** with preprocessing, normalization, and augmentation
- Built an automated, real-time ML pipeline that reduced manual intervention and improved scalability
- Achieved a **15–20% precision improvement** over the metaheuristic baseline

**Stack:** `Python` `PyTorch` `OpenCV` `U-Net`
